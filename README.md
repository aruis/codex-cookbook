# Codex Cookbook

一份面向团队与个人开发者的 Codex 实战手册，聚焦高频场景、可复制模板和稳定复用的协作方法。

## 怎么读

- 先看“道”：理解和 Codex 协作时最容易决定结果质量的底层判断。
- 再看“法”：确定任务该怎么组织，而不是直接堆提示词。
- 遇到具体局面时，看“术”：按不同场景选择打法。
- 需要马上复制时，用“器”：直接拿模板、反模式和通用补充句。

## 结构

- 道：底层判断
- 法：协作组织
- 术：局面打法
- 器：现成模板

## 适用方式

文档里的模板默认都是写给 Codex 的，可以直接复制，再按具体任务替换占位内容。

主文档见 [codex-cookbook.md](./codex-cookbook.md)。

## Slides

仓库内附带一份基于 reveal.js 的演示文稿，入口文件是 [docs/index.html](./docs/index.html)。

直接用浏览器打开即可；如果你想本地起一个静态服务，也可以在仓库根目录执行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000/docs/index.html`。
