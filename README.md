# Codex Cookbook

一份面向团队与个人开发者的 Codex 协作手册，重点沉淀高频场景下的方法、模板和可复用骨架。

主文档在 [codex-cookbook.md](./codex-cookbook.md)。

如果你第一次读，建议按这个顺序看：

- “道”：底层判断
- “法”：协作组织
- “术”：局面打法
- “器”：可直接复用的模板和补充句

## Slides

仓库内附带一份基于 reveal.js 的演示文稿，入口文件是 [docs/index.html](./docs/index.html)。

如果你想本地预览，可以在仓库根目录执行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000/docs/index.html`。
